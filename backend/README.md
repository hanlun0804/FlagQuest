
## Description

The FlagQuest backend is using the Nest.js TypeScript framework.

## Requirements

- Node.js

## Installation

Make sure you are in the `FlagQuest/backend` directory.
Run: 
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
